# Tic-Tac-Toe Web App

A minimalist, responsive Tic-Tac-Toe game built for fast, browser-based play.

- **Live Demo:** [https://tictactoe-rhrevgmu.edgeone.app/](https://tictactoe-rhrevgmu.edgeone.app/)
- **Features:** Classic 3x3 grid, local 2-player mode, and instant win detection.
- **Tech Stack:** Lightweight HTML, CSS, and JavaScript.
- **Deployment:** Optimized and hosted via the EdgeOne platform.
- **Objective:** Outmaneuver your opponent to get three marks in a row!
