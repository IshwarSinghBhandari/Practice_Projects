## 2. Modern Color Picker (Chrome Extension)
- **Features:** Screen-picking with EyeDropper API, hex/rgb/hsl conversion, and color history.
- **Tech Stack:** Tailwind CSS v4, Vanilla JavaScript, Chrome Extension Manifest V3.
- **How to Run Locally:** 
    1. Open Chrome and navigate to `chrome://extensions/`.
    2. Enable **Developer mode** in the top right.
    3. Click **Load unpacked** and select the `Color_Picker_Extension` folder.
- **AI-Assisted:** This project used AI for help in design and speed.
